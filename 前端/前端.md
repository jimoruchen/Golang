
table
align属性名，属性值为left center right，规定表格相对周围元素的对齐方式
border属性名，属性值1或"",规定表格是否有边框，默认为""，表示没有边框
cellpadding属性名，属性值像素值,单元格内容和边框之间的空白，默认为1
cellspacing属性名，属性值像素值,单元格和单元格之间的距离，默认为2
width属性名，属性值像素值或百分比,规定表格的宽度
height属性名，属性值像素值或百分比,规定表格的高度

thead
用于将表格中的表头行（通常是第一行）进行分组。它本身不会产生任何视觉变化，但具有重要的语义和结构意义。  

tr
代表“Table Row”（表格行）。每一个<tr>标签定义表格中的一行。

th
代表“Table Header”（表头单元格）。用于定义表格中的标题单元格。

tbody
用于将表格的主体内容（即数据行）进行分组。

td
代表“Table Data”（表格数据单元格）。用于定义表格中的标准数据单元格。

### 盒子宽度模式
给盒子设置宽高有两种模式。一种是 content-box，另一种是border-box。通常情况下，我们会使用border-box作为盒子的宽度模式，那么设置的width就是border+padding+content
```html
box-sizing: border-box;
```
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>盒模型</title>
    <style>
        *{
            box-sizing: border-box;
        }
        .box{
            width: 200px;
            height: 200px;
            padding: 20px;
            background: aqua;
            border: 10px solid red;
            margin: 20px;
        }
    </style>
</head>
<body>
<div class="box">内容</div>
</body>
</html>
```

### display
none：不显示

block：块盒，上下的margin会合并
独占一行：每个 block 元素会自动换行，前后有“换行效果”。
可设置宽高：width、height 完全生效。
上下左右 margin/padding 都有效：能正常撑开空间。

inline：行盒，width设置无效，两个盒子在一行不能设置宽高
不独占一行：多个 inline 元素可以在同一行内排列。
无法设置宽高：width 和 height 无效（由内容撑开）。
上下 margin/padding 无效（布局上）

inline-block：行内块，可以设置宽高
不独占一行：多个 inline-block 元素可以在同一行显示（像 inline）。
可设置宽高：width、height 有效（像 block）。
所有方向的 margin/padding 都有效：上下左右都能撑开空间。

<img src="https://s2.loli.net/2025/11/04/RAQqxHiyfwhr5vb.png" >

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>盒模型</title>
    <style>
        *{
            box-sizing: border-box;
        }
        .block{
            width: 300px;
        }
        .sub_block1{
            color: aqua;
            display: inline;
        }
        .sub_block2{
            color: red;
            display: inline;

    </style>
</head>

<div class="block">
```

## css定位

所谓定位，简单的来说就是将盒子定在某一个位置。所以定位其实也是按照规则进行摆盒子

值	描述
absolute	生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
fixed	生成固定定位的元素，相对于浏览器窗口进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
relative	生成相对定位的元素，相对于其正常位置进行定位。因此，“left:20” 会向元素的 LEFT 位置添加 20 像素。
static	默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）
sticky	粘性定位。根据用户的滚动位置进行定位。

### 相对定位

2. 之前的位置还在文档流中

### 粘性定位
粘性定位的元素是依赖于用户的滚动，在 position:relative 与 position:fixed 定位之间切换。

它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。

元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位。

占据原来的位置
以浏览器为参考
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>固定定位</title>
    <style>
        .d1{
            height: 500px;
            width: 100%;
            background: antiquewhite;
        }
        .d2{
            height: 40px;
            width: 40px;
            background: aqua;
            position: sticky;
            top: 100px;
        }
        .d3{
            height: 10000px;
            width: 100%;
            background: aliceblue;
        }
    </style>
</head>
<body>
<div class="d1"></div>
<div class="d2"></div>
<div class="d3"></div>
</body>
</html>
```

```css
to left 表示从右往左的方式进行渐变
to top left 表示从右下向左上进行渐变(top和left可以交换位置，下同)
to top right 表示从左下向右上进行渐变
to bottom left 表示从右上向左下进行渐变
to bottom right 表示从左上向右下进行渐变
```

## css中的flex布局
Flexbox 布局也叫 Flex 布局，弹性盒子布局。
它决定了元素如何在页面上排列，使它们能在不同的屏幕尺寸和设备下可预测地展现出来，更简便、完整、响应式地实现各种页面布局。
它的主要思想是使父元素能够调整子元素的宽度、高度、排列方式，从而更好的适应可用的布局空间。

### 容器属性

#### flex-direction
flex-direction 属性决定主轴的方向（即项目的排列方向）。

属性值	        含义
row	            默认值，主轴为水平方向（水平布局），起点在左端，从左向右排列
row-reverse	    主轴为水平方向（水平布局），起点在右端，从右向左排列
column	        主轴为垂直方向（垂直布局），起点在上沿，从上往下排列
column-reverse	主轴为垂直方向（垂直布局），起点在下沿，从下往上排列
```css
        .flex1{
            margin-top: 20px;
            display: flex;
            flex-direction: column;     /*默认是按行排*/
            /*flex-direction: column-reverse;*/
        }
```
<hr>

#### justify-content
justify-content属性定义了项目在主轴上的对齐方式。

属性值	        含义
flex-start	    默认值，左对齐
flex-end	    右对齐
center	        居中
space-between	两端对齐，项目之间的间隔都相等
space-around	每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍
space-evenly	均匀排列每个元素
```css
        .flex{
            display: flex;
            /*justify-content: space-between;*/
            /*justify-content: center;*/
            justify-content: space-evenly;
        }
```
<hr>

#### align-items
align-items属性定义项目在交叉轴上如何对齐。

属性值	    含义
flex-start	交叉轴的起点对齐
flex-end	交叉轴的终点对齐
center	    交叉轴的中点对齐
baseline	项目的第一行文字的基线对齐
stretch	    （默认值） 如果项目未设置高度或设为auto，将占满整个容器的高度
```css
        .flex2{
            margin-top: 20px;
            width: 500px;
            height: 400px;
            background-color: aqua;
            display: flex;
            flex-direction: row;
            justify-content: center;
            align-items: center;
        }
```
<hr>

#### flex-wrap
默认情况下，项目都排在一条线（又称"轴线"）上。flex-wrap属性定义，如果一条轴线排不下，如何换行。

属性值	        含义
nowrap	        默认值，表示不换行
wrap	        换行
wrap-reverse	换行，第一行在下方
```css
        .flex3{
            margin-top: 20px;
            width: 500px;
            height: 400px;
            background-color: aqua;
            display: flex;
            flex-direction: row;
            justify-content: start;
            align-items: start;
            flex-wrap: wrap;
        }
        .flex3 .item{
            width: 100px;
        }
```
<hr>

#### flex-flow
flex-flow 属性是 flex-direction 和 flex-wrap 属性的复合属性。

```css
/*flex-direction: row;*/
/*flex-wrap: wrap;*/
flex-flow: row wrap;
```

<hr>

#### align-content
属性和align-items一样，但是多了space-的设置

align-content适用于多行情况

<hr>

#### order
order属性用来定义项目的排列顺序。数值越小，排列越靠前，默认为 0 。

将盒子4排在前面
```css
        .flex4{
            margin-top: 20px;
            width: 500px;
            background-color: aqua;
            display: flex;
        }
        .flex4 .item{
            order: 10;
        }
```
```html
<div class="flex4">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item" style="order: 1">4</div>
</div>
```

<hr>

#### flex-grow
flex-grow属性定义项目的放大比例，默认为 0 ，即如果存在剩余空间，也不放大。

<hr>

#### flex-shrink
flex-shrink属性定义了项目的缩小比例，默认为 1 ，即如果空间不足，该项目将缩小。

<hr>

#### flex-basis
flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，
计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。
它的初始值是 auto，此时浏览器会检查元素是否设置了 width 属性值。
如果有，则使用 width 的值作为 flex-basis 的值；
如果没有，则用元素内容自身的大小。
如果 flex-basis 的值不是 auto，width 属性会被忽略。

<hr>

#### flex
flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

<hr>

#### align-self
align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，
表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

## css中的grid布局
Flex 布局是轴线布局，只能指定“项目”针对轴线的位置，可以看作是一维布局。
Grid 布局则是将容器划分成“行”和“列”，产生单元格，然后指定"项目所在"的单元格，可以看作是二维布局。Grid 布局远比 Flex 布局强大。
一般情况下，在处理某一行的元素的布局时使用flex布局；在处理多行的元素布局时使用grid布局。

### grid容器属性

#### grid-template-columns 

划分列数
固定宽度
```css
grid-template-columns: 40px 40px;
```
百分比
```css
grid-template-columns: 40% 60%;
```
比例
```css
grid-template-columns: 1fr 2fr;
```
重复
```css
grid-template-columns: repeat(5, 1fr);  /*5列，每列等宽*/
``` 
自动计算列
```css
grid-template-columns: repeat(auto-fill, 600px); /*每列固定600px，自动算多少列*/
grid-template-columns: repeat(auto-fit, 600px); /*每列固定600px，自动算多少列*/
```
auto
```css
grid-template-columns: 100px auto 100px;
```

```css
        .grid {
            width: 600px;
            background-color: #a84141;
            margin-top: 20px;
            display: grid;
            grid-template-columns: repeat(4, 1fr);
            row-gap: 20px;
            column-gap: 20px;
        }
        .grid .item{
            width: 100%;
            height: 40px;
            background-color: #247ad0;
        }
```
