标签	作用

##### table标签
```html
<table>
    <thead>
    <tr>
        <th>序号</th>
        <th>name</th>
    </tr>
    </thead>
    <tbody>
    <tr>
        <td>1</td>
        <td>张三</td>
    </tr>
    <tr>

        <td>李四</td>
```

table
align属性名，属性值为left center right，规定表格相对周围元素的对齐方式
border属性名，属性值1或"",规定表格是否有边框，默认为""，表示没有边框
cellpadding属性名，属性值像素值,单元格内容和边框之间的空白，默认为1
cellspacing属性名，属性值像素值,单元格和单元格之间的距离，默认为2
width属性名，属性值像素值或百分比,规定表格的宽度
height属性名，属性值像素值或百分比,规定表格的高度

thead
用于将表格中的表头行（通常是第一行）进行分组。它本身不会产生任何视觉变化，但具有重要的语义和结构意义。  

tr
代表“Table Row”（表格行）。每一个<tr>标签定义表格中的一行。

th
代表“Table Header”（表头单元格）。用于定义表格中的标题单元格。

tbody
用于将表格的主体内容（即数据行）进行分组。

td
代表“Table Data”（表格数据单元格）。用于定义表格中的标准数据单元格。

### 盒子宽度模式
给盒子设置宽高有两种模式。一种是 content-box，另一种是border-box。通常情况下，我们会使用border-box作为盒子的宽度模式，那么设置的width就是border+padding+content
```html
box-sizing: border-box;
```
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>盒模型</title>
    <style>
        *{
            box-sizing: border-box;
        }
        .box{
            width: 200px;
            height: 200px;
            padding: 20px;
            background: aqua;
            border: 10px solid red;
            margin: 20px;
        }
    </style>
</head>
<body>
<div class="box">内容</div>
</body>
</html>
```

### display
none：不显示

block：块盒，上下的margin会合并
独占一行：每个 block 元素会自动换行，前后有“换行效果”。
可设置宽高：width、height 完全生效。
上下左右 margin/padding 都有效：能正常撑开空间。

inline：行盒，width设置无效，两个盒子在一行不能设置宽高
不独占一行：多个 inline 元素可以在同一行内排列。
无法设置宽高：width 和 height 无效（由内容撑开）。
上下 margin/padding 无效（布局上）

inline-block：行内块，可以设置宽高
不独占一行：多个 inline-block 元素可以在同一行显示（像 inline）。
可设置宽高：width、height 有效（像 block）。
所有方向的 margin/padding 都有效：上下左右都能撑开空间。

<img src="https://s2.loli.net/2025/11/04/RAQqxHiyfwhr5vb.png" >

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>盒模型</title>
    <style>
        *{
            box-sizing: border-box;
        }
        .block{
            width: 300px;
        }
        .sub_block1{
            color: aqua;
            display: inline;
        }
        .sub_block2{
            color: red;
            display: inline;
        }
    </style>
</head>
<body>
<div class="block">
    <div class="sub_block1 inline">你好</div>
    <div class="sub_block2 inline">你好</div>
</div>
</body>
</html>
```

## css定位

所谓定位，简单的来说就是将盒子定在某一个位置。所以定位其实也是按照规则进行摆盒子

值	描述
absolute	生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
fixed	生成固定定位的元素，相对于浏览器窗口进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
relative	生成相对定位的元素，相对于其正常位置进行定位。因此，“left:20” 会向元素的 LEFT 位置添加 20 像素。
static	默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）
sticky	粘性定位。根据用户的滚动位置进行定位。

### 相对定位
1. 相对元素之前的位置
2. 之前的位置还在文档流中

```html
<!DOCTYPE html>

            height: 30px;
            background: aqua;
            position: relative;
            left: 30px;
            top: 30px;

    </style>
</head>
<body>

</body>
```

### 粘性定位
粘性定位的元素是依赖于用户的滚动，在 position:relative 与 position:fixed 定位之间切换。

它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。

元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位。

占据原来的位置
以浏览器为参考
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>固定定位</title>
    <style>
        .d1{
            height: 500px;
            width: 100%;
            background: antiquewhite;
        }
        .d2{
            height: 40px;
            width: 40px;
            background: aqua;
            position: sticky;
            top: 100px;
        }
        .d3{
            height: 10000px;
            width: 100%;
            background: aliceblue;
        }
    </style>
</head>
<body>
<div class="d1"></div>
<div class="d2"></div>
<div class="d3"></div>
</body>
</html>
```
